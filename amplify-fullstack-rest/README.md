# AWS Amplify Full-Stack REST Application

A full-stack serverless web application built using **React** and **AWS Amplify**, with a **REST API** backend powered by **API Gateway** and **AWS Lambda**.

This project demonstrates how to design, deploy, and manage cloud infrastructure using **Infrastructure as Code (IaC)** while integrating it with a modern frontend.
